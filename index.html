Glosario de propiedades CSS

Abajo verás la lista de propiedades, su descripción y ejemplos de valor. También tenemos ejemplos visuales que usan esas mismas propiedades dentro del <style>.

Glosario

Propiedad Descripción Ejemplo de valor
color Color del texto. blue, #0000FF, rgb(0, 0, 255)
border Abreviatura para ancho, estilo y color del borde. 2px solid black
border-radius Redondea las esquinas del borde. 10px, 50%
margin Espacio exterior alrededor del elemento. 10px 5px 10px 5px (arriba, derecha, abajo, izquierda)
padding Espacio interior entre el borde y el contenido. padding-top: 10px;, padding: 12px;
text-align Alineación horizontal del texto dentro del elemento. center, justify
background-size Tamaño de la imagen de fondo. cover, contain
text-decoration Efectos de subrayado, tachado u otras decoraciones del texto. underline, line-through, none
float Alinear un elemento a la izquierda o derecha, permitiendo que el texto lo rodee. left, right
